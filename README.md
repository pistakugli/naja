# NAJA - Agent Zero + Claude AI

Fork of [Agent Zero](https://github.com/frdel/agent-zero) with Claude AI tools.

## Quick Start
```bash
git clone https://github.com/pistakugli/naja.git
cd naja
./install_claude_skills.sh  # Optional
./naja-integrated
```

## What's New
- 12 custom NAJA tools (`instruments/custom/naja-tools/`)
- 16 Claude AI skills (via release)
- Unified launcher (`naja-integrated`)
